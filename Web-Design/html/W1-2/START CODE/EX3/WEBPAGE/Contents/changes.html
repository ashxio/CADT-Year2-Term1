ការផ្លាស់ប្ដូរទីតាំង

        នៅឆ្នាំ១៩៨៥ វត្តវរិន្ទព្រែកលាបបានផ្ដល់ដីនៅទល់មុខនោះ(ទីតាំងសព្វថ្ងៃ)ឲ្យកសាងសាលារៀនដោយព្រះចៅអធិការវត្តលោកយល់ថា ចង់ឲ្យអនុវិទ្យាល័យ និងបឋមសិក្សាស្ថិតនៅដាច់ដោយឡែកពីគ្នា។ ម្យ៉ាងទៀត បើយោងតាមសម្ដីរបស់លោកគ្រូសុវន និងលោកគ្រូសោភ័ណ បានឲ្យដឹងថា ការដែលព្រះចៅអធិការបានផ្ដល់ដីមកឲ្យសាលាព្រោះលោកចង់ឱ្យសាលានៅដាច់ដោយឡែកពីវត្ត និងបានសល់កន្លែងសម្រាប់វត្តអភិវឌ្ឍន៍ដូចជាកសាងកុដ សាលាបាលីជាដើម។

សណ្ឋានដីមុខសាងសង

        ពីដើមឡើយ ដីដែលវត្តផ្ដល់ឱ្យសាលានេះជាដីហោដ្ឋានដែលវត្តបានទុកចោល ហើយជាទីតាំងមួយដែលប្រជាជននាំគ្នាសូមព្រះសង្ឃបញ្ចុះសព និងផ្នែកខាងអគារចែកការ(JICA)សព្វថ្ងៃនេះ ពីមុនមកជាបឹងធំមួយ ឯដីខាងក្រោយសាលាបង្អស់ហួសតារាងបាល់កាលដើមឡើយជាបន្ទាយខ្មែរលើ(ភ្នង)ដែលគេកាប់ប្រឡាយភ្ជាប់ពីទីនោះទៅដល់ព្រែកតាសេក។ ដោយដីសំបូរទៅដោយបឹង និងផ្នូរជាច្រើន ទើបសាលាបានធ្វើការរៀបចំគាស់ផ្នូរ និងចាក់ដីបំពេញបន្ថែម។ ការកសាងមួយនេះត្រូវបានប្រជាជនមួយចំនួនតវ៉ាដែរ តែពេលលោកគ្រូ អ្នកគ្រូ រួមទាំងចៅអធិការបានបកស្រាយ ពួកគាត់ក៏ព្រមស្របតាម។ កាលនោះសាលាគ្មានលុយគ្រប់គ្រាន់សម្រាប់ជួលឡានកាយដីចាក់បំពេញទេ ក៏បានលក់ដីឲ្យទៅគេសម្រាប់ថ្លៃស្នួលឡានកាយដី ដោយគេកាយដី២ឡានដែលមួយឡានជាដីរបស់យើង មួយឡានទៀតជាដីគេ។ ការធ្វើបែបនេះប្រើពេលយូរពេកទើបសាលាក៏សម្រេចលក់ដីមួយផ្នែកតូចដើម្បីបានដីមកបំពេញបឹង និងសាងសង់សាលា។ចំពោះដីនៅទីតាំងថ្មីនេះមានទំហំប្រហែលបណ្តោយ២១៧ម៉ែត្រ ទទឹង៨០ម៉ែត្រ។

ការចាប់ដំណើរការនៅទីតាំងថ្មី

នៅឆ្នាំ១៩៨៨ ដីត្រូវបានចាក់បំពេញរួចរាល់ សាលាក៏ឆាប់ផ្ដើមកសាងអគារមុនគេដែលធ្វើពីថ្មមានចំនួ២ជាន់ ១០បន្ទប់ដែលជាអំណោយរបស់ក្រសួង។ នៅឆ្នាំ១៩៩២ វិទ្យាល័យព្រែកលៀបបានកាត់បញ្ចូលមកក្នុងរាជធានីវិញដែលស្ថិតនៅខាងលិចផ្លូវលេខ៦ សង្កាត់ជ្រោយចង្វារ រាជធានីភ្នំពេញ។ ពេលនោះគ្រប់គ្នាហៅឈ្មោះថា អនុវិទ្យាល័យព្រែកលៀប។ ក្រោយមកនៅឆ្នាំ២០០៥ បានក្លាយទៅជាវិទ្យាល័យព្រែកលៀបក្រោមការគ្រប់គ្រងរបស់នាយក វង់ សាវុន។

ការអភិវឌ្ឍក្រោយផ្លាស់ប្តូរមកកាន់ទីតាំងថ្មី

        ក្រោមការគ្រប់គ្រងរបស់លោកគ្រូនាយក វង់ សាវន់ មកកាន់ទីតាំងថ្មីបានធ្វើឱ្យមានការអភិវឌ្ឍជាច្រើនដូចជា៖

  • ការសាងសង់របង២០១ និងបណ្ណាល័យមួយខ្នង(២០០២)
  • កសាងអគារសិក្សាមួយខ្នងមានពីរជាន់ស្មើនឹង១០បន្ទប់ដែលជាអគារខ(២០០៥)
  • កសាងអគារមួយខ្នងទៀតមានពីរជាន់មានស្មើ១០បន្ទប់ដែរនៅខាងត្បូងឈាងខាងលិចដែលជាអគារគ(២០០៨)
  • ចាក់ខ្សាច់បំពេញរណ្តៅ រៀបការ៉ូឡានៅមុខអគារគ និងធ្វើរបងប្រវែង៤១១ម៉ែត្រពីកើតទៅជើង(២០១១)
  • កសាងរបងខាងមុខប៉ែកខាងកើត ប្រវែង៤២ម៉ែត្រ(២០១២)

        នៅកំឡុងឆ្នាំ២០១៤ដេលស្ថិតក្រោមការគ្រប់គ្រងរបស់លោកគ្រូ ឆុំ សុភក្ដិ ដែលលោកបានខ្នាសខ្នែងបង្កើតសម្ធផលជាច្រើន។ នៅឆ្នាំ២០១៤ និង២០១៥គឺបានសង់របងខាងជើងបន្តបានប្រវែង១១៣.៥ម៉ែត្រ និងបានសង់ផ្សារសាលា(canteen)អោយមានលក្ខណៈសមរម្យមួយខ្នងដែលមានទំហំ១៥មx២៧ម។ នៅចុងឆ្នាំ២០១៦គឺបានរៀបការ៉ូឡានៅខាងមុខអគារគ។

ផលវិបាកនៃការផ្លាស់ប្ដូរមកទីតាំងថ្មី

        ការផ្លាស់ប្ដូរទីតាំងថ្មីបានធ្វើឲ្យយើងជួបនូវវិបត្តិកង្វះថវិកា។ ដើម្បីកសាងដីដែលនៅតំបន់ថ្មីដែលមានបឹង យើងត្រូវចំណាយលុយច្រើនក្នុងការកាយដីលុបបំបាត់បឹង។ ទន្ទឹមនឹងនេះដែរ ទីតាំងថ្មីរបស់សាលាគឺជួបប្រទះនឹងការជនលេចទឹកនូវរដូវវស្សាដោយសារតែប្រឡាយខាងក្រោយនោះ និងស្ថិតនៅទាបជាងផ្លូវ។ ក្រោយមកដោយមានការចូលរួមជួយពីសំណាក់ក្រសួងសប្បុរសជន លោកគ្រូអ្នកគ្រូ និងប្រជាជនទើបបញ្ហាទាំងអស់នេះត្រូវបានដោះស្រាយហើយសាលាក៏បានដំណើរការរលូន។

កំណែទម្រង់ទៅលើកម្នវិធីសិក្សា

        ក្នងឆ្នាំ២០១៧-២០១៨ វិទ្យាល័យព្រែកលៀបត្រូវបានក្រសួងអបរំ យុវជន និងកីឡា ដាក់ឱ្យអនុវត្តគោលនយោបាយទី១៤នៃកំណែទម្រង់វិស័យអប់រំគឺអនុវត្តកម្មវិធីសាលារៀនជំនាន់ថ្មី។ គេបានប្ដូរឈ្មោះសាលាទៅជា សាលារៀនជំនាន់ថ្មីវិទ្យាល័យព្រែកលៀប។

ការអភិវឌ្ឍក្រោយកំណែទម្រង់

        នៅដើមឆ្នាំ២០១៧ សាលាបានរៀបការ៉ូឡា ជួសជុលរបងដែកនៅខាងក្នុងបរិវេណកន្លែងផ្ញើកង់-ម៉ូតូ និងតារាងបាល់ទះដោយលើកខឿនចាក់ដីបំពេញ ព្រមទាំងតម្រង់សរសរឡើងវិញ។ នៅឆ្នាំ២០១៨ ជប៉ុនបានជួយសាងសង់អគារ១ខ្នង កំពស់៣ជាន់ មាន២១បន្ទប់ដែលជាជំនួយរបស់រាជរដ្ឋាភិបាល។ បន្ទាប់មកក៏កែលម្អអគារ២ខ្នងតាមស្តង់ដាសាលារៀនជំនាន់ថ្មី ដោយជួសជុលបំពាក់ឧបករណ៍ទំនើបៗនានា បង្កើតបណ្ណាល័យទំនើបបែបសតវត្សទី២១ និងជួសជុលអាគារគទៅតាមស្តង់ដាសាលារៀនជំនាន់ថ្មី។ សាលាក៏បានរៀបការ៉ូឡាខាងមុខអាគារចែកកា និងខាងក្រៅរបងជាប់ផ្លូវជាតិ(ជំនួយរបស់ប្រធានមន្ទីរសាធារណការនិងដឹកជញ្ជូនរាជធានីភ្នំពេញ)។ សាលាAISបានឧបឋមបង់ថ្មអង្គុយចំនួន១០ ហើយមន្ទីរសាធារណការ និងដឹកជញ្ជូនរាជធានីភ្នំពេញសហការជាមួយអង្គការJICA បានចាក់ដីបំពេញវាលខាងក្រោយបន្ថែមទៀត។ នៅឆ្នាំ២០២០ បានសាងសង់អគារប្រជុំថ្មី(ប្តូរទីតាំងទៅផ្នែកខាងមុខវិញ)។ នៅឆ្នាំ២០២១ សាលាបានសាងសង់តារាងបាល់ទាត់មួយកន្លែងដែលមានទំហំ20មx40ម ដែលជាជំនួយរបស់អង្គការការជប៉ុន។ វិទ្យាល័យព្រែកលៀបមានអាគារសិក្សាចំនួន៥ខ្នង និង៥៣បន្ទប់។
