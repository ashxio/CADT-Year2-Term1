ការស្ដារសាលាឡើងវិញ(១៩៧៩)

        ក្រោយឆ្នាំ១៩៧៩ក្រោយពីរបបកម្ពុជាប្រជាធិបតេយ្យបានដួលរលំ តាមតំបន់បានចាប់ផ្តើមសាងនិងជុសជុលកន្លែងនីមួយៗសារជាថ្មីម្ដងទៀត ហើយគ្រានោះផងក៏ជាពេលដែលអនុវិទ្យាល័យតែមួយគត់ដែលស្ថិតក្នុងខេត្តកណ្តាលបានងើបមុខសារជាថ្មីម្ដងទៀត។ តាមការផ្ដល់បទសម្ភាសរបស់លោកគ្រូ យក់ សោភណ្ឌ បានជម្រាបប្រសាស៏ឱ្យដឹងថា “សាលានេះត្រូវបានកសាងតាំងពីមុនឆ្នាំ១៩៧៩ទៅទៀតឯណោះតែយើងពុំអាចសន្និដ្ឋានឱ្យប្រាកដចំពោះឆ្នាំកសាងបានឡើយព្រោះគ្មានឯកសារណានៅសល់អាចបញ្ជាក់ឱ្យបានច្បាស់លាស់នោះទេ។ យើងអាចសន្និដ្ឋានបានសាលាកើតមុនឆ្នាំ១៩៧៩ដោយសារតែការបន្សល់នៅអគារសិក្សា២ខ្នង (ថ្មមួយ ឈើមួយ) ជាភស្តុតាង។ ត្បិតតែយើងមិនអាចបកស្រាយនៅតថៈនោះបានមែនតែយើងក៏អាចមានឯកសាដែលបរិយាយអំពីការងើបជាថ្មីនៃសាលាដែរ។

       នៅឆ្នាំ១៩៨១ សាលាបានកសាងជាថ្មីក្រោមការជ្រុមជ្រែងរបស់ព្រះចៅអធិការព្រះនាម ម៉ាសារ៉េត បន្ទាប់មានលោកគ្រូ ឃុត មឿង ដែលមកឈរមើលការខុសត្រូវក្នុងការកសាងសាលា។ ក្រោយមកទើបមានលោកស្រី ហុង អេង មកមើលជំនួសលោកម្ដង។ ក្នុងការកសាងក៏មានប្រជាជនចូលរួមជួយផងដែរ”។

ការសាងសង់

       សាលាត្រូវបានកសាងដោយឈើដែលបានមកពីឈើដែលគេរើផ្ទះចាស់ និងសល់ពីពពតដែលជាសសរផ្ទះប្រជាជនជាដើម។ ឈើខ្លះមានការពុកផុយ ឈើខ្លះក៏មានលក្ខណៈល្អប្រណិតផងដែរ។ កាលនោះក៏មានសិស្សបានធ្វើវិភាគទាន៤០៛ ព្រមទាំងមានប្រជាជនមកជួយក្នុងការកសាងលើកឈើជាដើម។ ការកសាងបានបង្កើតឱ្យមានវត្តមាន កម្រិតសិក្សាទី២ បន្ថែមទៀត។

ដំណើរការសាលាក្រោយ

        មុននឹងដំណើរការសាលា គេបានកសាងអគារពីឈើមួយបន្ថែមទៀតដែលមាន៤បន្ទប់ក្នុងមួយអគារ និងបានកសាងទីចាត់ថ្មីមួយធ្វើពីឈើព្រមទាំងរបងសាលា

        នៅពេលដំណើការបង្រៀន ក្នុងមួយថ្នាក់មាន៣៣នាក់(គ្មានឯកសារបញ្ជាក់ឆ្នាំឌំណើរការច្បាស់លាស់)។ ពេលបើកដំណើរការដំបូងគឺមានតែ២ថ្នាក់ប៉ុណ្ណោះ រីឯការជ្រើសរើសគ្រូគឺបានមកពីលោកគ្រូអ្នកគ្រូដែលជាបញ្ជវ័នមុនសម័យខ្មែរក្រហមដែលនៅសេសសល់បានស្ម័គ្រចិត្តមកបង្រៀន ហើយកាលនោះតាមប្រសាសន៍លោកគ្រូ យក់ សោភ័ណ បានឱ្យដឹងថា “កាលនោះគេយកអ្នកចេះច្រើនបង្រៀនអ្នកចេះតិច ហើយគ្មានមុខវិជ្ជាជាគោលច្បាស់លាស់ណាមួយឡើយ គ្រូម្នាក់ត្រូវបង្រៀនសិស្សបីបួនមុខវិជ្ជាតែឯងតាមដែលខ្លួនចេះ ឬមួយថ្នាក់បង្រៀនដោយគ្រូម្នាក់ ហើយគ្រូគឺឡើងឋានៈតាមសិស្ស(បើសិស្សរៀនទី៤គ្រូបង្រៀនទី៤ ហើយពេលសិស្សឡើងទី៥គ្រូត្រូវបង្រៀនទី៥) តែមុននឹងទៅបង្រៀនបាន គ្រូត្រូវទៅបំប៉នបន្ថែមពីមួយទៅមួយខែកន្លះសិន”។ បើនិយាយពីការសិក្សាជំនាន់នោះ បឋមសិក្សាដែលជាថ្នាក់កម្រិតសិក្សាទី១(មានពីទី១-៥) ទើបប្រឡងកាលេស ឡើងកម្រិតសិក្សាទី២(អនុវិទ្យាល័យមានចាប់ពីទី៦-៩)។ ថ្នាក់ទី៥ស្មើនឺងទី៧ឥឡូវ ក្រោយពេលបញ្ចប់កម្រិតសិក្សាទី២ ត្រូវទៅប្រឡងឌីប្លូមនៅសាលាគៀនស្វាយ និងសាលាដីឥដ្ឋហើយបន្តកម្រិតសិក្សាទី៣នៅតាខ្មៅ ។
