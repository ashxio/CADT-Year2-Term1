សេចក្ដីផ្ដើម

បក្សពួកខ្មែរដែលទំនោរកុម្មុយនីស្ដបានផ្ដួលរលំរបបសារធាណរដ្ឋដោយអាវុធហើយបានបង្កើតរបបថ្មីមួយទៀតឈ្មោះថា “កម្ពុជាប្រជាធិបតេយ្យ” ដែលជារបបផ្តាច់ការ និងតម្រូវឱ្យមានភាពស្មើគ្នា គ្មានអ្នកមានឬអ្នកក្រឡើយ។ នៅក្នុងសម័យនោះត្រូវបានដឹកនាំដោយ សាឡុត ស។ កាលនោះគេបានប្រើឧបាយក្នុងការជំលៀសប្រជាជនចេញពីភ្នំពេញទៅតាមបណ្ដាលខេត្ត។ សំណង់អគារធំៗ វត្ត សាលារៀន រូបបដិមា រូបិយវត្ថុ នឹងវត្ថុមានតម្លៃនានា ត្រូវបានគេបំផ្លាញអស់ ធ្វើឱ្យទីក្រុងខូចខាតហេដ្ឋារចនាសម្ព័ន្ធស្ទើរគ្រប់ទិសទី ដោយឡែកសាលារៀនមួយដែលស្ថិតក្នុងវត្តវារិន្តព្រែកលៀបដែលជាសាលាបឋមសិក្សាដំបូងបង្អស់ក្នុងខេត្តកណ្តាល មានឈ្មោះថា អនុវិទ្យាល័យព្រែកលៀបនឹងបឋមសិក្សាព្រែកលៀប ដែលកាលនោះគច្រើនហៅថា សាលាកម្រិតសិក្សាទី១ ក៏បានរងការខូចហេដ្ឋារចនាសម្ព័ន្ធ និងបានជួបនឹងវិបត្តិសង្រ្គាមនៅក្នុងសម័យកម្ពុជាប្រជាធិបតេយ្យហើយត្រូវបានទុកចោលអស់រយៈពេលជាច្រើនឆ្នាំរហូតមកដល់ឆ្នាំ១៩៧៩ទើបត្រូវបានគេជុសជុលឡើងវិញ។
